## Installation

```bash
npm i chop-array
```

## Usage

```js
import chopArray from 'chop-array'

chopArray(['a', 'b', 'c', 'd', 'e', 'f'], 2)
//=> [['a', 'b'], ['c', 'd'], ['e', 'f']]

chopArray(['a', 'b', 'c', 'd', 'e', 'f', 'foo'], 3)
//=> [['a', 'b', 'c'], ['d', 'e', 'f'], ['foo']]
```
